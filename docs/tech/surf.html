          # 移动端教程 (v2rayNG)

          # 1. 添加订阅地址

          打开 v2rayNG 客户端 -> 点击左上角 ≡ 菜单图标 -> 选择 订阅设置 -> 点击右上角 + 号 -> 输入 备注（如：免费节点）并粘贴上方的 订阅地址 -> 点击右上角对钩保存。

          # 2. 更新订阅列表

          返回主页面 -> 点击右上角 ⋮ 菜单图标 -> 选择 更新订阅。稍等片刻，主页面将自动拉取并显示节点列表。

          # 3. 连接性测试 (真连接延迟)

          点击右上角 ⋮ 菜单图标 -> 选择 测试全部配置真连接 -> 等待进度条完成。节点右侧显示数值（如 256ms）即代表该节点可用。

          # 4. 筛选可用节点

          建议再次点击右上角 ⋮ 菜单 -> 选择 删除无效配置(先测试)，这样列表中只会保留当前可用的服务器节点。

          # 5. 选择节点并启动

          从列表中点击选择一个延迟较低的可用节点 -> 点击右下角的 V 字形圆形图标。图标由灰色变为绿色即表示连接成功。

          # 6. 验证连接状态

          打开浏览器访问 Google (opens new window)，若能正常打开则表示翻墙成功。如果连接缓慢，请尝试切换至其他数值较小的节点。

          # 7. 停止科学上网

          再次点击右下角的 绿色圆形图标 -> 图标变回灰色，即表示已安全断开连接。

          # 电脑端教程 (v2rayN)

          # 1. 配置订阅设置

          双击任务栏小图标打开 v2rayN 主界面 -> 点击上方菜单栏的 订阅 -> 订阅设置 -> 在弹窗中点击 添加 -> 填写 备注 并粘贴 订阅地址 -> 务必勾选 启用 并点击 确定。

          # 2. 更新订阅节点

          点击主菜单栏的 订阅 -> 选择 更新订阅 (不通过代理)。此时，程序会从订阅地址拉取最新的免费节点列表。

          # 3. 批量测试节点延迟

          在主界面节点列表中，按 Ctrl + A 全选所有节点 -> 按 Ctrl + R 进行真连接延迟测试。查看 真连接延迟 列，数值越小连接速度越快。

          # 4. 选择节点并开启代理

          选中一个延迟较低的节点 -> 按 Enter 回车键激活。接着右键点击任务栏托盘小图标 -> 选择 系统代理 -> 自动配置系统代理。此时图标颜色会由蓝色变为红色。

          # 5. 验证是否翻墙成功

          在浏览器中尝试打开 Google (opens new window)，若能顺畅访问即表示配置成功。

          # 6. 灵活切换节点

          若当前节点变慢，可以右键托盘图标 -> 服务器 -> 直接从列表中切换。或在主界面选中新节点后按 Enter 键切换。

          # 7. 停止科学上网

          右键点击任务栏托盘小图标 -> 选择 系统代理 -> 清除系统代理。图标恢复为蓝色，即表示已关闭代理。
